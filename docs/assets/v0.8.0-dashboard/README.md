# v0.8.0 dashboard screenshots

Captured for README/release polish on 2026-05-11 using the local Hermes Vault dashboard served from repo source with a temporary demo runtime under `/tmp`.

Capture method:
- A throwaway `HERMES_VAULT_HOME` was created for the session.
- The vault was seeded with fake/demo credentials only.
- The dashboard was served on `127.0.0.1` with its normal token-guarded local session URL.
- Headless Google Chrome captured the dashboard views through the browser, with the intro skipped for clean static screenshots.

No real credentials, tokens, OAuth material, or production vault data were used. The screenshots show only redacted dashboard metadata and fake/demo service inventory.
